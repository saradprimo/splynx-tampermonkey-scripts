Splynx Tampermonkey Scripts

Install any of these scripts with one click. Make sure Tampermonkey is installed in your browser. Scripts will auto-update when a new version is released.

  • ETC Calculator Install ETC Calculator with categories, plan filtering, Mates Rates logic, and Fibre Starter always $0 ETC.
  • Activity Feed Visual Separator Install Adds borders and backgrounds to activity items and always shows full text.
  • ONT Enquiry Button Install Shows ONT Enquiry button only on Services tab. Supports single or multiple PIIDs.
  • ONT Enquiry Notifier & Summary Install Full ONT summary, collapsible sections, handover/power symbols, notifier button, connected devices, and GPON RX/TX classification.
