Splynx Tampermonkey Scripts

Click a script link below to install it directly in Tampermonkey. Make sure you have Tampermonkey installed in your browser.

  • Splynx ETC Calculator

    Calculates Early Termination Charges (ETC) based on contract start date and plan. Applies plan-specific rules, including capped amounts and Fibre Starter always $0 ETC.

  • Activity Feed Visual Separator

    Adds borders, background, and shadows to activity items in tickets dashboard in Splynx. Ensures full text is always visible and highlights admin system items.

  • ONT Enquiry Button

    Shows a single ONT Enquiry button on the Services tab. Auto-selects single PIID or provides a stable dropdown for multiple PIIDs for fast diagnostics.

  • ONT Enquiry Notifier & Summary

    Provides a full ONT summary with collapsible sections, handover & power status symbols, connected devices with vendor info, scrollable summary, and real-time notifications for test completion or failures.
